
## What this repository demonstrates
Public portfolio archive for superconducting-device R&D context, YBCO Josephson diode/ratchet analysis, cryogenic I–V interpretation, critical-current asymmetry, rectification, output-power and efficiency analysis, measurement-chain reasoning, and cryogenic/instrumentation documentation.

## Research context
This repository supports superconducting-device and applied-physics portfolio related to YBCO Josephson diode/ratchet work.

## Relevant topics
- YBCO Josephson diode / Josephson ratchet
- cryogenic I–V analysis
- critical-current asymmetry
- AC/noise rectification analysis
- output voltage, output power, and efficiency interpretation
- DAQ/ADC and measurement-chain context
- voltage/current acquisition
- FFT/PSD noise-analysis context
- cryogenic dipstick documentation

## Claim boundary
This repository does not claim sole fabrication ownership, sole He-FIB operation, full cleanroom process ownership, or completed final direct noise-driven Josephson-ratchet operation.
