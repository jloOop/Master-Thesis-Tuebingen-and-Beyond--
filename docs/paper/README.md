## Repository relation

The repository contains historical notebooks, scripts, instrumentation notes,
and thesis-era material related to:

- IVC characterization;
- critical-current asymmetry;
- sine-drive Josephson-ratchet rectification;
- Gaussian/noise-drive rectification calculations;
- noise-chain preparation;
- cryogenic measurement hardware and DC dipstick design.
