# DC cryogenic dipstick design documentation

This folder documents a DC cryogenic dipstick and compatible PCB design used in
the Tübingen superconducting-device measurement context.

The file `designed_dc_dipstick.pdf` contains visual documentation of:

- Inventor CAD renders of the dipstick assembly;
- mechanical drawings and dimensions for custom parts;
- fabricated brass/copper components;
- PCB routing and channel layout;
- assembly photographs of the dipstick head, PCB, wiring, and sample region.

This material supports the hardware/instrumentation side of the Josephson
diode/ratchet project. It is not direct figure-reproduction data for the paper;
it is a hardware-design and measurement-infrastructure record.
