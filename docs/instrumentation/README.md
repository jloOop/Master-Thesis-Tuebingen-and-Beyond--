# Instrumentation and cryogenic measurement documentation

This directory collects documentation for the laboratory measurement chain used
around the YBCO Josephson diode/ratchet project.

The material here is documentation-level context, not raw numerical data. It
includes noise-chain analysis, cryogenic dipstick design, PCB/mechanical
documentation, and setup-level notes relevant to Josephson-ratchet measurements.

## Contents

- `noise-characterization/`: external noise identification and noise-budget
  documentation for ADC, voltage amplifier, current source, generator, resistor,
  and dipstick configurations.
- `dc-dipstick/`: DC cryogenic dipstick design, PCB routing, mechanical drawings,
  CAD renders, and assembly photographs.

For machine-readable summaries extracted from the noise slides, see:

- `data/processed/noise-budget/`
