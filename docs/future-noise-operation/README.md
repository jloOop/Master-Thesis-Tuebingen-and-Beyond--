# Future noise-operation preparation

This folder contains follow-up setup-characterization material prepared by
Alireza Jozani for future direct noise-driven Josephson-ratchet operation.

The document is not a reproduction file for the published Physical Review
Applied paper. It is a continuation of the thesis-era Josephson-ratchet work,
focused on preparing and diagnosing the experimental measurement chain required
for controlled noise-operation studies.

## Main document

* `noise_preparation_setup_characterization_may2025.pdf`  
  Setup-by-setup noise-preparation slides from May 2025. The document inspects
  the measurement chain component by component, including ADC, voltage amplifier,
  current source, waveform generator, resistors, and dipstick configurations.

## Scope

The slides summarize:

* ADC baseline checks;
* voltage-amplifier and current-source noise contributions;
* generator-noise characterization;
* resistor thermal-noise comparisons;
* shorted-dipstick and connected-dipstick tests;
* gain-bandwidth and roll-off limitations;
* possible low-noise amplifier strategies for future measurements.

This material should be read as setup-preparation and measurement-chain
characterization for future Josephson-ratchet noise-operation work, not as raw
measurement data and not as a claim that the final follow-up experiment was
completed in this repository.
