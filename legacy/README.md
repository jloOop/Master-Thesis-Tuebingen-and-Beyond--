# Legacy material

This folder preserves original notebooks and scripts from the thesis-era workflow.

| Folder | Purpose |
|---|---|
| `original_notebooks/` | Original notebooks retained for traceability. |
| `original_root_scripts/` | Original root-level scripts retained for archival context. |

## Interpretation

Legacy files are not intended as a polished software interface. They may depend on original local paths, lab hardware, instrument configuration, or historical folder layouts. Use the clean notebooks and documentation folders as the primary public entry points.
